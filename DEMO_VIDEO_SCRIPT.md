- **Resolution:** 1920x1080 (Full HD minimum)
- **Frame Rate:** 60 FPS
- **Bitrate:** High quality (10+ Mbps)
- **Audio:** Clear microphone, no background noise
- **Software:** OBS Studio or Camtasia

### During Recording
- Slow, deliberate movements
- Pause after each action (1 second)
- Highlight important UI elements with mouse
- Ensure all text is readable
- Show loading states completing
- Demonstrate smooth animations

- Add background music (low volume, no copyright)

- Add transitions between sections
- Color grading for consistency
